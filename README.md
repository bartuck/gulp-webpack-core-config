# Gulp and Webpack - Core Configuration
HTML, SCSS, ES6+, Gulp, Webpack (Import and Export Modules), Translations.

## Getting Started
`$ npm install`

## How to use
Work with the following directories and files:
- src/index.html
- src/assets
- src/js
- src/scss
- src/translations

## Commands
Just build and finish

`$ gulp build`

Build and serve

`$ gulp serve`

Build, serve and watch for changes

`$ gulp watch`
